# Product-Management
[![Forks][forks-shield]][forks-url]
[![MIT License][license-shield]][license-url]


A mini project on Product Management using php. 

## Features
  * Homepage
  * Registtration
  * Login
  * Dashboard
  * Product table with Actions
    * Add Product
    * Update Product
    * Delete Product
  * User Profile
    * View Profile
    * Edit Profile
  * Change Profile Picture
  * Change Password
